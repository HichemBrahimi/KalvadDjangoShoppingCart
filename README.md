Ce projet porte sur le développement d'un site Web . Il fournit à l'utilisateur une liste des différents produits disponibles à l'achat dans le magasin. Pour la commodité des achats en ligne, un panier est fourni à l'utilisateur. Après la sélection de la marchandise, il est envoyé pour le processus de confirmation de commande. Le système est implémenté à l'aide du framework Web Python Django.

afficher les produits, les clients pourront sélectionner des catalogues et sélectionner des produits, et pourront supprimer des produits de leur panier en spécifiant la quantité de chaque article. 
la sécurité grâce à l'utilisation d'identifiants de connexion et de mots de passe, afin qu'aucun utilisateur non autorisé ne puisse accéder à votre compte.

Technologies utilisées dans le projet :
Framework Django et base de données SQLite fournis par défaut avec Django.

Compétences requises pour construire le projet :
Connaissance de Python et des bases de Django Framework.
